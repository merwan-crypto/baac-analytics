# Documentation du projet

Ce dossier contient de la documentation pédagogique vous permettant de comprendre plus facilement / rapidement le code qui est mis à votre disposition :

- `structure.md` : Description de la structure du dépôt et rôle de chaque fichier/dossier.
- `mvc.md` : Rappels sur l'architecture MVC, avec un mapping vers ce projet.

Il vous est recommandé de vous approprier le code au maximum, de le comprendre parfaitement avant de le modifier et de le personnaliser. En effet, la structure et le design qui y sont construits sont indicatifs et peuvent être entièrement refaits selon vos goûts.
